# Xself Home - React Native App

A beautiful furniture shopping app built with React Native.

## Features

- 🏠 Home - Product grid with categories
- 🔍 Discover - Search & filter (Wayfair-style)
- 🛒 Cart - Shopping cart with quantity
- ❤️ Saved - Wishlist
- 👤 Account - Profile & settings
- 📱 Product Detail - Full product info

## Quick Start

```bash
# Install dependencies
npm install

# Run on iOS Simulator
npx react-native run-ios

# Or run on Android
npx react-native run-android
```

## Tech Stack

- React Native 0.73
- React Navigation 6
- Native Stack
- Bottom Tabs

## Project Structure

```
src/
├── screens/      # Screen components
├── components/   # Reusable components
├── data/         # Mock data
└── utils/        # Utilities
```

## Screens

1. **Home** - Product listing with search, categories, member banner
2. **Discover** - Wayfair-style visual discovery
3. **Product Detail** - Full product info with add to cart
4. **Cart** - Shopping cart with totals
5. **Account** - Profile and menu
