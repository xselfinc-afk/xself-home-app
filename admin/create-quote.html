Xself Concierge — Create Special Offer

Internal agent tool. Quotes are server-validated: never below $50 and never less than 50% of list price.

Setup (one-time per browser)
Supabase dashboard → Settings → API.
Used only for product search (read-only).
Shared secret. Saved to localStorage on this browser only.
Stored alongside the quote in support_quotes.created_by_agent for audit.

Product

Customer

Must match the email the customer is signed in with in the app.

Offer

Dollars. $50 floor, 50% list maximum discount.
Default 168 (7 days). Maximum 720 (30 days).
If supplied, customer receives a system message in chat. The redeem token is never included in that message.

Once created, the customer will see the offer the next time they open Concierge for this product on a signed-in account that matches the email above. The redeem token stays on the server.
